# lp-src — LP用の元写真を入れる場所

このフォルダに、LP（`lp.html`）で使う**元の写真ファイルをドラッグして入れてください**。
名前は変えなくてOK。入れたらチャットで「入れたよ」と教えてください。

入れてもらった写真を、私（Claude）が内容を見て下のスロットに割り当て、
`public/lp-assets/` に分かりやすい名前でコピー → `lp.html` の参照を書き換え → git にコミットします。
これで写真がコードと一緒に管理され、二度と行方不明になりません。
