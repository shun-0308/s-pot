# 水彩アート地図の作り方(AI生成ワークフロー)

県ページの地図を、大三島マップのような水彩イラストにするための手順。

## 仕組み

- このフォルダの `01_guide.png`〜`47_guide.png` は各県の**形状ガイド**(JIS都道府県コード順)
- AI画像生成でこのガイドを下絵にして水彩地図を作り、**`public/maps/{県コード}.png`** に置くだけで、アプリが自動でその絵を下敷きに表示します(例: 東京都 → `public/maps/13.png`)
- 記録の地名ラベル・ピンはアプリが上に重ねるので、**絵に文字を入れない**こと
- 画像がない県は今のカーキのシルエットで表示されます(フォールバック)

## 生成のコツ

1. ChatGPT / Gemini などに **ガイド画像を添付**して、下のプロンプトを使う
2. できた画像とガイドを見比べ、**形が崩れていたら「形状は添付画像に正確に合わせて」と再生成**
3. 画像の縦横比はガイドと同じまま保存(アプリ側で枠いっぱいに貼るため)
4. ファイル名を県コード(01〜47)にして `public/maps/` へ

## プロンプトテンプレート(コピペ用)

```
添付した地図シルエットの形状・位置・縦横比を正確に保ったまま、
手描き水彩のイラストマップに仕上げてください。

スタイル:
- やさしい水彩。陸は淡い黄緑〜若草色、海は淡い水色に手描きの波線
- 古い旅ノート・フィールドノートの雰囲気。紙は生成り(#F5F0E6)
- 山地には小さな山のスケッチ、川は細い水色の線
- 輪郭は柔らかい茶色の細いペン線
- 文字・地名・記号は一切入れない(アプリ側で重ねるため)
- 影や3D効果は不要。フラットな水彩

出力: 添付画像と同じ縦横比、2000px程度
```

## 備考

- まず東京都(13)・愛媛県(38)など、よく使う県から試すのがおすすめ
- 全47枚を一気に作らなくてOK。置いた県から順に絵に切り替わります
- 海外の国ページ用は将来対応(必要になったら国ごとのガイドも書き出せます)
